# REG Open Source Service Area — docs website

A Jekyll site using the [minima](https://github.com/jekyll/minima) theme published to GitHub Pages at https://alan-turing-institute.github.io/hut23-open-source-sa/.

## Pages

| File | URL |
|---|---|
| `index.md` | `/` — home page |
| `hacksessions.md` | `/hacksessions/` |
| `ospo.md` | `/ospo/` |
| `contributions.md` | `/contributions/` |
| `events.md` | `/events/` |

## Local development

```bash
cd docs
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000/hut23-open-source-sa/.
